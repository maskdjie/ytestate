0. Persiapan Dasar (Sekali Seumur Hidup)

A. Menghubungkan Otak AI (Gemini Key)

Fitur Magic AI tidak akan berfungsi tanpa API Key Gemini. Ini gratis dan kapasitasnya besar.

  1. Buka situs resmi Google AI Studio.
  2. Login dengan akun Google biasa. Klik tombol biru "Create API key".
  3. Copy kode unik yang muncul (biasanya diawali huruf AIza...).
  4. Buka aplikasi > Tab PENGATURAN > Tempel di kolom Gemini API Key > Klik Simpan.

B. Membuat Kunci API YouTube (JSON)

Ini adalah 'tiket' aplikasi agar bisa mengakses channel Anda secara resmi.

  1. Buka Google Cloud Console. Buat New Project.
  2. Masuk ke APIs & Services > Library. Cari dan Enable: YouTube Data API v3 & YouTube Analytics API.
  3. Masuk ke OAuth consent screen. Pilih External. Isi App Name bebas. Klik Save & Continue sampai selesai.
  4. PENTING: Klik tombol "PUBLISH APP" di halaman OAuth agar statusnya In Production (Token jadi awet).
  5. Masuk ke Credentials > + Create Credentials > OAuth client ID. Pilih tipe Desktop App.
  6. Download file JSON tersebut. Masukkan ke folder secrets di direktori aplikasi.

2. Fitur Live Streaming (High Performance)

üöÄ TEKNOLOGI ENGINE V11

Gunakan Ghost Mode FFmpeg yang sangat hemat CPU (CPU < 5%). Mendukung durasi siaran hingga 24 jam non-stop.

A. Menyiapkan Live Streaming Baru

  1. Buka Tab LIVE STREAMING. Klik tombol ‚ûï Add Stream.
  2. Target Channel: Pilih channel yang ingin digunakan.
  3. Video Sumber: Pilih file video di komputer. Gunakan video durasi lama (misal 1 jam) agar lebih stabil.
  4. Metode AUTO API: Masukkan judul atau gunakan ‚ú® AI. Tentukan durasi (misal 11 jam).
  5. üîÅ Mode Loop: Centang ini agar video diputar berulang-ulang sampai durasi live berakhir.

B. Memahami "Silent Mode" (Metode Hantu) üëª

Ini adalah detail paling krusial untuk keamanan akun Anda:

  • Apa itu? Jika dicentang, jadwal Live TIDAK AKAN muncul di YouTube sekarang.
  • Keamanan: Mencegah laporan spam/flagging sebelum live dimulai.
  • Cara Kerja: Jadwal hanya ada di database lokal RDP Anda. Saat jam tayang tiba, wlive.exe baru akan melakukan 'Booking' ke YouTube API dan langsung memulai siaran saat itu juga.

  • Batasan Thumbnail: File gambar wajib di bawah 2 MB. Jika lebih, YouTube API akan menolak dan status menjadi GAGAL.

  • Multi-Worker: wupload.exe dan wlive.exe bisa berjalan bersamaan tanpa saling mengganggu.
  • Anti-Banned: Gunakan fitur Humanizer di Tab Pengaturan untuk memberikan jeda acak antar upload agar pola tidak terbaca bot.

    • Shorts: Video vertikal < 3 menit otomatis masuk Shorts.
    • Thumbnail: Wajib di bawah 2MB (JPG/PNG).
